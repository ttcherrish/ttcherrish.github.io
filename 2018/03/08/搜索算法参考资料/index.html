搜索算法参考资料

Mar 8, 2018 | 阅读

前言

本文整理了搜索算法相关知识以及一些参考资料，不断完善中。

基础知识

基础知识是几乎每天的工作都要用到的，基础的内容需要熟练掌握。

  • 编程语言

    • python
      • 《简明python教程》（A Byte of Python ）
        • python3：英文原版 中文版
        • python2: 中文版
        • 建议使用python3，python2.7将于2020年1月1日终止支持
    • java
      • 会使用一种IDE，如: IntelliJ IDEA/Eclipse等
    • 理解面向对象的基本知识

  • Linux

    • 熟练使用一种编辑器：Vim/Emacs等
    • 基本命令
      • Linux命令五分钟
    • Shell脚本
      • Shell脚本学习

  • Git

    • Pro Git, Second Edition
  • Hadoop
    • MapReduce、streaming、hive等
  • 正则表达式

搜索相关

  • 理解倒排索引
    • 理解倒排索引的基本原理
    • 玩一些开源工具，了解倒排索引是如何应用的，如：lucene/ ES
  • Learning To Rank
    • Ranklib
      • LTR算法的开源包
      • 跑通demo数据
      • 阅读引用的论文，理解算法原理
    • 经典书籍
      • Learning to Rank for Information Retrieval pdf
      • Learning to Rank for Information Retrieval and Natural Language Processing pdf

机器学习

  • 经典书籍
    • 机器学习 周志华
    • 统计学习方法 李航
    • Pattern Recognition and Machine Learning pdf
    • deep learning（花书) 英文原版 中文版
  • 公开课
    • Machine Learning
      • Andrew Ng 老师Coursera上的经典入门课程
    • 台大林轩田老师的课程
      • 相比于Ng的课程，属于进阶课程
      • 机器学习基石（Machine Learning Foundations）
        • Coursera：上 下
        • 百度网盘
      • 机器学习技法（Machine Learning Techniques）
        • Coursera：课程已下线
        • 百度网盘
    • 谷歌的机器学习速成课
    • Deep Learning Specialization
      • Coursera
      • 网易云课堂
      • 网易云课堂没有习题，对于初学者，习题与视频同等重要，建议Coursera
    • 动手学深度学习
      • 课程汇总
      • 非常好的中文课程，基于MXNet的gluon接口
      • 不建议初学者把gluon作为一个深度学习框架使用，流行度不够，遇到问题不易找到解决方法，可以使用相类似的PyTorch
    • fast.ai
      • YouTube视频搬运到 百度网盘
      • 这几门深度学习课程内容重合度较高，着重学习一门即可

  • python开源包：numpy，pandas，sklearn等
  • JStorm
